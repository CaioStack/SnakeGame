🐍 **Cyberpunk Snake Game**

Um clássico reinventado com estética cyberpunk neon.
Este projeto é uma versão moderna do famoso Snake Game, desenvolvido com HTML, CSS e JavaScript puro, trazendo uma interface futurista inspirada em neon, tecnologia e ambientes cyberpunk.

🎮 **Funcionalidades**
Movimentação fluida da cobra
Sistema de pontuação
Crescimento da snake ao comer alimentos
Estilo visual cyberpunk/neon
Game Over ao colidir
Interface simples e responsiva

📂 **Estrutura do Projeto**

```
📁 SnakeGame

 ├── game.js # Lógica do jogo (movimento, colisão, pontuação)
 
 ├── index.html # Estrutura do jogo (tela, elementos, layout)
 
 └── style.css # Visual e estilo cyberpunk/neon do jogo

```
